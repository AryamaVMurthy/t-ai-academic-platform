Lecture Notes

Introduction to Neural Networks

Neural networks are a set of algorithms, modeled loosely after the human brain, that are designed to recognize patterns.

Key Components

Input Layer, Hidden Layers, Output Layer, Weights, Activation Functions, and Loss Functions.

Forward Propagation

Process where input data is passed through the neural network to get an output.

Backpropagation

Algorithm used to train neural networks by calculating gradients and updating weights to minimize loss.

  • Calculate error/loss at output
  • Compute gradients using chain rule
  • Update weights backwards through network

Activation Functions

ReLU, Sigmoid, Tanh, and Softmax functions and their use cases.

Training Process

Batch sizes, epochs, learning rates, and optimization algorithms.

What's the difference between ReLU and sigmoid activation?

ReLU (Rectified Linear Unit) outputs the input directly if positive, otherwise 0. It's computationally efficient and helps with vanishing gradient problems.

Sigmoid squishes values between 0 and 1, making it useful for binary classification but prone to vanishing gradients for deep networks.
