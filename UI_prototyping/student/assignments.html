What's the difference between Dijkstra's and Bellman-Ford algorithms?

Dijkstra's algorithm finds the shortest path in a graph with non-negative weights, running in O(E log V) time with a priority queue.

Bellman-Ford works with negative weights too and can detect negative cycles, but runs slower at O(VE) time.

For this assignment, focus on Dijkstra's since all weights are positive.
